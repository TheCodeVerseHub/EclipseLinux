# Security Policy

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Instead, use GitHub's private vulnerability reporting for this repository:
https://github.com/TheCodeVerseHub/EclipseLinux/security/advisories/new

If you can’t use GitHub advisories, contact the security team via the organization’s listed security contact.

## Supported Versions

Security fixes are typically applied to the latest `main` branch and the most recent release.
